Trago dicas.

Para desenvolver bem é preciso muita prática e habilidade. Para adquirir prática e habilidade é preciso investir tempo, ler livros, ver vídeos e conteúdos como os que estão disponíveis abaixo para uma boa fixação do que você pode precisar em breve.

Fique atento as dicas e guarde-as em alguma pasta no seu dispositivo para não perder de vista, assim sempre que precisar vai ter em mãos.

Manter a sua área de trabalho organizada agora precisa ser uma das suas rotinas de desenvolvedor. Perder os arquivos em pastas que aleatórias te farão perder tempo.

Classes rápidas

No HTML para criar uma classe rápido basta usar o ponto antes do nome da classe. Se forem duas ou mais classes separe as classes por pontos e vai criar uma tag class com os seus respectivos seletores.

Paleta de cores

Use as paletas de cores do Color Adobe gratuitamente para fazer as cores de forma harmoniosa dentro do site. É só pesquisar no google AdobeColor ou Color Adobe e você vai acessar um site com várias formas de combinar cores.

Formulário sem back-end

E se eu te disser que tem como fazer envio de formulários sem precisar saber PHP, Javascript e nenhuma outra linguagem de programação (back-end)? Existem algumas ferramentas grátis na internet que possibilitam isso, uma delas é o FormSubmit e eu tenho um vídeo explicando como faz tudo lá no YouTube. Confira!

Emmet Abbreviation para agilizar

Para encapsular qualquer tag ou conteúdo dentro de outra tag usando o Emmet Abreviation agilisa o processo de desenvolvimento.

Para usar esse recurso no VsCode você precisa usar o atalho Ctrl + Shft + P, selecionar o opção Emmet:Wrap with Abbreviation e nela digita a tag que deseja encapsular.

Hospedagem grátis

Você sabia que tem como hospedar seu site totalemnte de grça?

Isso mesmo que você leu. A Netlify oferece muito espaço para você hospedar os seus projetos de forma 100% free e ainda mais um monte de recursos que tem siponível lá no site da Netlify.com.

Eu tenho um tutorial muito bom pra te ajudar a fazer a sua hospedagem sem gastar nada no meu canal. Para conferir essa super dica acessa o vídeo aqui!

Tenha animações no seu site

Não saber JS não vai mais te impedir de ter um site cheio de animações em scroll. Isso poquê exite uma livraria liberada no GitHub que você pode acessar e implementar no seu HTML sem esforço.

As animações de scroll da Aos Animate são simples, bonitas e tem algumas variedades para escolher se quer subindo, descendo, em fip e outras mais.
